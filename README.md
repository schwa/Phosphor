# Phosphor

A [Shadertoy](https://www.shadertoy.com)-like playground for Metal compute shaders, using AI to generate and modify shaders.

Built on [MetalSprockets](https://github.com/schwa/MetalSprockets).

## Requirements

- macOS 27 (Golden Gate)

## Shader generation

- **On-device** generation works but isn't very good (small context window).
- **Private Cloud Compute** is not yet enabled.
- **Claude** requires your own Anthropic API key (set it in Settings → Models).
